VARIABLE NAMES: LEGAL & ILLEGAL

Code:
var name, age, gender;

Code:
// legal variable names
var name, AGE, _gender, $nationality, religion123;
// illegal variable names
var 123name, @AGE, .gender, !nationality, reli gion;

Rules for naming JS variables

  1. Variable names can only contain letters, numbers, underscore(_) and Dollar sign($), For example $my_1stVariable
  2. Variables must begin with a letter, underscore(_) or Dollar sign($). For example $name, _name or name
  3. Variable names are case sensitive
  4. Variable names should not be JS Keywords
